Rellena el formulario y descarga el archivo Markdown listo para subir a la carpeta posts/.

Flujo de publicación:
  1. Rellena el formulario y pulsa Generar Markdown
  2. Descarga el archivo .md generado
  3. En Replit, sube ese archivo a la carpeta posts/
  4. Copia el bloque JSON de la pestaña "JSON para blog-posts.json"
  5. Abre blog-posts.json y pega el bloque al principio de la lista
  6. Pulsa Deploy / Publish
